# Kumo_Blog
PHP+MySQL+Bootstrap+AJAXを駆使している個人ブログ

## 機能一覧
* ログイン
* 記事一覧
* 記事新規
* 記事削除
* 記事更新

### 引き続き機能作成予定
* プロフィール修正
* パスワード修正
* 記事検索
* タグ・カテゴリーで記事検索
* 記事へのいいね！
* 訪問数による記事の並び替え
